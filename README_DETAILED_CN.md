# Dayflow - 智能时间追踪与生产力分析系统

## 核心功能

### 智能屏幕录制
- 自动录制屏幕活动，支持高画质压缩
- 智能暂停/恢复：检测到用户闲置时自动暂停录制，恢复活动时自动继续
- 可配置录制参数：画质、帧率、间隔等
- 自动视频切片管理：按时间间隔切片并存储

### 活跃度监测系统
- 多维度活动检测：键盘输入、鼠标移动、窗口焦点
- 智能闲置判断：允许思考期间的无操作时间
- 状态机检测逻辑：精确判断用户工作状态
- 自动暂停/恢复：长时间无操作自动暂停，检测到活动自动继续

### AI视频分析
- 批量视频处理：定时扫描待分析视频切片
- 智能活动识别：自动识别工作、学习、娱乐、休息等活动类型
- 效率评分：基于活动内容评估工作效率（0-100分）
- 自动生成活动卡片：包含时间、类别、标题、效率评分等信息

### 健康提醒系统
- 工作时长提醒：连续工作超过阈值后提醒休息
- 娱乐时长警告：长时间娱乐活动提醒注意平衡
- 活动切换提醒：活动类型切换时智能提醒
- 优先级管理：重要提醒优先显示，不被覆盖
- 提醒持久化：未处理的提醒保存到数据库，重启后仍然有效

### 每日总结功能
- 自动生成每日活动总结报告
- 识别关键活动（如深度工作、长时间会议等）
- 展示每日效率评分变化趋势
- 按类别统计时间使用情况
- 提供个性化改进建议

### 时间轴管理
- 可视化时间轴：直观展示一整天的活动分布
- 卡片式展示：每个活动以卡片形式呈现，包含详细信息
- 手动编辑：支持修改活动类别、标题、效率评分等
- 日期切换：快速切换到不同日期查看历史记录

### 数据统计分析
- 多维度统计：总时长、平均效率、深度工作时长等
- 日期范围查询：支持按天、周、月、自定义范围查询
- 分类统计：按活动类别统计时间和效率
- 效率分析：识别高效和低效时间段
- 数据导出：支持导出统计数据

### 设置管理
- 录制设置：调整录制参数（画质、帧率、间隔等）
- 监测设置：配置活跃度检测参数（闲置阈值、检测窗口等）
- 提醒设置：自定义健康提醒规则和优先级
- 分析设置：调整AI分析参数（批量时长、扫描间隔等）
- 界面设置：主题切换、语言设置等

## 数据隐私
- 所有数据本地存储，完全掌控个人数据
- 视频切片自动清理，可配置保留天数
- 无云端数据上传

## 配置参数

### 录制配置
```python
RECORD_FPS = 5                    # 录制帧率
RECORD_BITRATE = 500000           # 视频比特率
CHUNK_DURATION_SECONDS = 900      # 切片间隔（15分钟）
MAX_CHUNK_AGE_DAYS = 7            # 视频保留天数
```

### 活跃度监测配置
```python
STOP_CHECK_INTERVAL = 60          # 停止检测间隔（秒）
STOP_DETECTION_DURATION = 30      # 停止检测持续时间（秒）
RESUME_WAIT_DURATION = 30         # 恢复等待持续时间（秒）
RESUME_DETECTION_DURATION = 30    # 恢复检测持续时间（秒）
```

### 健康提醒配置
```python
WORK_DURATION_THRESHOLD = 90      # 工作提醒阈值（分钟）
ENTERTAINMENT_DURATION_THRESHOLD = 60  # 娱乐提醒阈值（分钟）
CHECK_INTERVAL = 60               # 检查间隔（秒）
WORK_PRIORITY = 10                # 工作提醒优先级
ENTERTAINMENT_PRIORITY = 5        # 娱乐提醒优先级
```

### 分析配置
```python
BATCH_DURATION_MINUTES = 15       # 批次时长（分钟）
ANALYSIS_INTERVAL_SECONDS = 60    # 扫描间隔（秒）
```
