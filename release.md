## 🚀 快速开始

### 安装方法

1. **下载 zip 包**
   ```bash
   点击上方下载链接获取 Dayflow-2026.5.6-windows.zip
   ```

2. **解压到任意目录**
   ```bash
   选择任意目录（如 D:\Dayflow\）进行解压
   ```

3. **运行 Dayflow.exe**
   ```bash
   双击 Dayflow.exe 启动程序
   ```

4. **保留 _internal 文件夹**
   ```bash
   ⚠️ 请勿删除 _internal 文件夹，它是程序运行所必需的依赖文件
   ```

### 首次使用

1. 启动程序后，点击左侧 **⚙️ 设置**
2. 配置 API 信息：
   - **API 地址**：默认为 `https://open.bigmodel.cn/api/paas/v4`（智谱AI）
   - **API 地址**：Coding Plan `https://open.bigmodel.cn/api/coding/paas/v4`（智谱AI coding plan）
   - **API Key**：填入你的 API 密钥
   - **模型名称**：默认为 `glm-4-flash-250414`（免费模型）
3. 点击 **测试连接** 验证配置
4. 点击 **保存配置** 完成设置
5. 点击 **▶ 开始录制** 开始使用

## 🆘 常见问题

### Q1: 程序启动失败？
**A:** 请确保 `_internal` 文件夹与 `Dayflow.exe` 在同一目录下，并且没有被删除或移动。

### Q2: API 连接失败？
**A:** 检查以下几点：
- API 地址是否正确（默认：https://open.bigmodel.cn/api/paas/v4）
- API Key 是否有效
- 网络连接是否正常
- 是否有防火墙或安全软件拦截

### Q3: 分析结果不准确？
**A:** 可以尝试：
- 切换到更强大的模型（如 glm-4-plus）
- 检查录屏帧率设置
- 确保窗口信息能正常获取

### Q4: 如何切换模型？
**A:** 
1. 打开 **设置** → **API 配置**
2. 修改 **模型名称** 字段
3. 点击 **测试连接** 验证
4. 点击 **保存配置**

### Q5: 如何备份数据？
**A:**
1. 打开 **设置** → **数据管理**
2. 点击 **📤 导出数据**
3. 选择要导出的日期
4. 选择保存位置
