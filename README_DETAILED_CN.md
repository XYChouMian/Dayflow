# Dayflow - 智能时间追踪与生产力分析系统

## 核心功能

### 智能屏幕录制
- 自动录制屏幕活动，支持低码率压缩
- 智能暂停/恢复：检测到用户闲置时自动暂停录制，恢复活动时自动继续
- 可配置录制参数：切片时长、帧率间隔等
- 自动视频切片管理：按时间间隔切片并存储（默认每60秒一个切片）

### 活跃度监测系统
- 多维度活动检测：键盘输入、鼠标移动、窗口焦点
- 智能闲置判断：允许思考期间的无操作时间（滑动窗口机制）
- 状态机检测逻辑：精确判断用户工作状态
- 自动暂停/恢复：长时间无操作自动暂停，检测到活动自动继续
- 可配置检测参数：闲置阈值、思考窗口、检测窗口时长等

### AI视频分析
- 批量视频处理：定时扫描待分析视频切片（默认每60秒扫描一次）
- 智能活动识别：自动识别工作、学习、娱乐、社交、休息等活动类型
- 效率评分：基于活动内容评估工作效率（0-100分）
- 自动生成活动卡片：包含时间、类别、标题、效率评分、应用/网站信息等
- 支持卡片合并：相似活动可自动合并

### 健康提醒系统
- 工作时长提醒：连续工作超过阈值后提醒休息（默认90分钟）
- 娱乐时长警告：长时间娱乐活动提醒注意平衡（默认60分钟）
- 活动切换提醒：活动类型切换时智能提醒
- 优先级管理：重要提醒优先显示，不被覆盖
- 提醒持久化：未处理的提醒保存到数据库，重启后仍然有效
- 冷却时间：两次提醒之间的最小间隔（默认15分钟）
- 可通过系统托盘显示提醒通知

### 灵感记录功能
- 快速记录想法和灵感
- 支持分类：灵感、想法、待办
- 支持添加备注标签
- 可编辑和删除记录
- 按日期查看灵感记录
- 灵感内容可作为每日总结的素材

### 每日总结功能
- 自动生成每日活动总结报告（AI生成）
- 识别关键活动（如深度工作、长时间会议等）
- 展示每日效率评分变化趋势
- 按类别统计时间使用情况
- 提供个性化改进建议
- 支持手动编辑和补充总结内容
- 包含事件总结和灵感总结两部分

### 周总结功能
- 自动生成近7日周总结报告（AI生成）
- 汇总一周的活动模式和效率变化
- 展示周维度的时间分配统计
- 识别一周的高效时段和低效时段
- 提供周维度改进建议
- 支持手动编辑和补充总结内容

### 时间轴管理
- 可视化时间轴：直观展示一整天的活动分布
- 卡片式展示：每个活动以卡片形式呈现，包含详细信息
- 手动编辑：支持修改活动类别、标题、效率评分等
- 删除功能：可以删除不需要的活动卡片
- 日期切换：快速切换到不同日期查看历史记录
- 支持查看任意日期的活动记录

### 数据统计分析
- Web仪表盘导出：生成交互式HTML报告，包含图表和数据可视化
- 多维度统计：总时长、平均效率、深度工作时长、活动数量等
- 日期范围查询：支持今日、昨日、本周、本月、自定义范围查询
- 分类统计：按活动类别统计时间和效率，饼图展示
- 效率分析：识别高效和低效时间段
- 趋势图：展示时间维度上的效率变化
- 热力图：展示一天中不同时段的活动分布
- 周对比：本周与上周的对比分析
- 数据导出：支持导出为JSON格式，便于备份和迁移

### 设置管理

#### API配置
- API地址配置：支持OpenAI兼容接口
- API Key管理：安全存储API密钥
- 模型选择：选择AI分析模型
- 测试连接：验证API配置是否正确

#### 录制设置
- 自动暂停开关：启用/禁用智能自动暂停功能
- 切片时长配置：调整视频切片间隔（默认60秒）
- 最小切片数量：配置批次分析的最小切片数
- 自动删除录屏：分析完成后自动删除视频切片节省磁盘空间

#### 健康提醒设置
- 工作时长阈值：连续工作多少分钟后提醒（默认90分钟）
- 娱乐时长阈值：连续娱乐多少分钟后提醒（默认60分钟）
- 冷却时间：两次提醒之间的最小间隔（默认15分钟）
- 启用/禁用提醒功能

#### 活跃度监测设置
- 停止检测间隔：检测闲置的检查间隔（默认60秒）
- 停止检测持续时间：判断为闲置需要持续的时间（默认30秒）
- 恢复等待持续时间：进入休息状态后的等待时间（默认30秒）
- 恢复检测持续时间：判断恢复活跃需要持续的时间（默认90秒）
- 思考窗口：允许的无操作时间（默认5分钟）

#### 界面设置
- 主题切换：支持暗色/亮色主题
- 自动保存主题偏好

#### 数据管理
- 修改存储位置：可自定义数据存储路径，支持数据迁移
- 手动清理数据：手动清理非今日的录屏文件
- 导出数据：导出选定日期的数据为JSON文件
- 导入数据：从JSON文件导入活动数据和灵感记录

#### 自启动设置
- 开机自启动：设置开机自动运行
- 启动模式：启动时最小化到托盘或显示主窗口

#### 更新设置
- 版本检查：检查是否有新版本
- 自动下载：后台下载更新包
- 多源下载：支持多个镜像源加速下载
- 自动安装：下载完成后一键安装更新

### 系统托盘功能
- 最小化到托盘：不占用任务栏
- 托盘菜单：快速开始/停止录制、显示/隐藏窗口、退出应用
- 托盘通知：健康提醒通过系统托盘显示

### 主题系统
- 暗色主题：适合夜间使用
- 亮色主题：适合白天使用
- 主题切换：一键切换主题
- 主题持久化：自动保存用户偏好
- 所有组件支持主题切换

## 数据隐私
- 所有数据本地存储，完全掌控个人数据
- 视频切片可配置自动清理
- 可手动清理历史录屏文件
- 无云端数据上传（仅调用配置的AI API进行分析）
- 数据存储路径可自定义
- 支持数据备份和迁移

## 配置参数

### 录制配置
```python
RECORD_FRAME_INTERVAL = 3.0  # 每帧间隔秒数（约0.33 FPS）
CHUNK_DURATION_SECONDS = 60   # 切片间隔（60秒）
VIDEO_BITRATE = "500k"       # 视频比特率
VIDEO_CODEC = "libx264"      # 视频编码格式
```

### 活跃度监测配置
```python
STOP_CHECK_INTERVAL = 60       # 停止检测间隔（秒）
STOP_DETECTION_DURATION = 30   # 停止检测持续时间（秒）
RESUME_WAIT_DURATION = 30      # 恢复等待持续时间（秒）
RESUME_DETECTION_DURATION = 90 # 恢复检测持续时间（秒）
```

### 健康提醒配置
```python
HEALTH_REMINDER_WORK_THRESHOLD = 90  # 工作提醒阈值（分钟）
HEALTH_REMINDER_ENTERTAINMENT_THRESHOLD = 60  # 娱乐提醒阈值（分钟）
HEALTH_REMINDER_COOLDOWN = 15       # 提醒冷却时间（分钟）
HEALTH_REMINDER_CHECK_INTERVAL = 300000  # 检查间隔（毫秒，5分钟）
```

### 分析配置
```python
BATCH_CHUNK_COUNT = 10             # 每个批次的切片数量
ANALYSIS_INTERVAL_SECONDS = 60     # 扫描间隔（秒）
```

### API配置
```python
API_BASE_URL = "https://open.bigmodel.cn/api/paas/v4"  # API地址
API_KEY = ""  # API密钥
API_MODEL = "glm-4-flash-250414"  # AI模型
API_TIMEOUT = 300.0  # API请求超时时间（秒）
```

- 模板引擎：Jinja2
